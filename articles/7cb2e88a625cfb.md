# IP固定化
Debian Bookworm 以降のバージョンは NetworkManager を使ってネットワークを設定する。

## 手順

### 1. イーサネットに設定された connection があるか確認
```bash
$ nmcli connection show | grep -i ethernet

Wired connection 1  00000000-0000-0000-0000-000000000000  ethernet  eth0
```
- `Wired connection 1`：コネクション名
- `00000000-0000-0000-0000-000000000000`：UUID
- `ethernet`：種別
- `eth0`：デバイス名

### 2. 固定アドレスにするIPv4アドレスを指定
```bash
$ sudo nmcli connection modify "Wired connection 1" ipv4.addresses "192.168.100.20/24"
```

### 3. 接続を down/up して設定を反映
```bash
$ sudo nmcli connection down "Wired connection 1"
$ sudo nmcli connection up "Wired connection 1"

Connection 'Wired connection 1' successfully deactivated (D-Bus active path: /org/freedesktop/NetworkManager/ActiveConnection/2)
```

### 4. 反映されているか確認
```bash
$ ip address show eth0

2: eth0: <BROADCAST,MULTICAST,UP,LOWER_UP> mtu 1500 qdisc mq state UP group default qlen 1000
    link/ether 00:00:00:00:00:00 brd ff:ff:ff:ff:ff:ff
    inet 192.168.100.2/24 brd 192.168.100.255 scope global noprefixroute eth0
       valid_lft forever preferred_lft forever
    inet 192.168.100.20/24 brd 192.168.100.255 scope global secondary dynamic noprefixroute eth0
       valid_lft 14386sec preferred_lft 14386sec
    inet6 0000::0000:0000:0000:0000/64 scope link noprefixroute
       valid_lft forever preferred_lft forever
```

### 5. DHCPでの割り当てを止める
```bash
$ sudo nmcli connection modify "Wired connection 1" ipv4.method manual
```

### 6. dnsサーバーを設定
```bash
$ sudo nmcli connection modify "Wired connection 1" ipv4.dns "192.168.100.254"
```

### 7. デフォルトゲートウェイを設定
```bash
$ sudo nmcli connection modify "Wired connection 1" ipv4.gateway "192.168.100.254"
```

### 8. 適用
```bash
$ sudo nmcli connection down "Wired connection 1"
$ sudo nmcli connection up "Wired connection 1"
```

### 9. 再度確認
```bash
$ ip address show eth0

2: eth0: <BROADCAST,MULTICAST,UP,LOWER_UP> mtu 1500 qdisc mq state UP group default qlen 1000
    link/ether 00:00:00:00:00:00 brd ff:ff:ff:ff:ff:ff
    inet 192.168.100.20/24 brd 192.168.1.255 scope global noprefixroute eth0
       valid_lft forever preferred_lft forever
    inet6 0000::0000:0000:0000:0000/64 scope link noprefixroute
       valid_lft forever preferred_lft forever
```

## コマンドまとめ
一般ユーザー権限下で行うものとする。
```bash
nmcli connection show | grep -i ethernet
sudo nmcli connection modify "Wired connection 1" ipv4.addresses "192.168.100.20/24"
sudo nmcli connection modify "Wired connection 1" ipv4.gateway "192.168.100.254"
sudo nmcli connection modify "Wired connection 1" ipv4.dns "192.168.100.254"
sudo nmcli connection modify "Wired connection 1" ipv4.method manual
sudo nmcli connection down "Wired connection 1" && sudo nmcli connection up "Wired connection 1"
ip address show eth0
```
